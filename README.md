## 📦 项目列表

### 1️⃣ [LayerMind](./LayerMind/) - Photoshop AI 图层智能命名/分类插件

一个基于 AI 技术的 Photoshop 图层智能管理工具，能够自动识别图层内容并生成具有语义的名称，同时根据内容自动创建分类分组。

**核心功能：**
- ✅ 一键智能命名：AI 自动识别图层内容，生成精准中文名称
- ✅ 自动分类前缀：【文字】【素材】【效果】【背景】等
- ✅ 图层自动分组：按内容类型、视觉层级自动分组
- ✅ 图层快速检索：关键词搜索，快速定位图层

**技术栈：** C++ / Qt 6 + ExtendScript (JSX) + 百度 AI 图像识别 API

---

### 2️⃣ [AI 图片工具箱](./ai_image_toolbox.py) - Python + Streamlit 图片处理工具

一个功能强大的 AI 图片工具箱，包含图片分类整理和物体标记两大功能。

**核心功能：**
- ✅ AI 图片分类整理：自动识别图片内容并分类（风景/美食/宠物/人物/文档/物品/植物）
- ✅ AI 水印/杂物标记：自动检测物体并用红色矩形框标记位置
- ✅ Streamlit 可视化界面：简洁美观的 Web 界面

**技术栈：** Python 3 + Streamlit + requests + Pillow + 百度 AI 图像识别 API
